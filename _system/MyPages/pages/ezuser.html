What is ezUser?
  • ezUser is the world’s easiest way to add user registration and logins to your website.
  • ezUser is a single PHP script that you include on your website. One line of PHP code will add user login to any page on your site.
  • ezUser is simple to use; the default settings will work for your site.
  • ezUser is powerful and flexible. Its visual style and the way it works can be customized to match your needs.
  • ezUser is free software. Go ahead and try it; there’s no cost, no trial period, no limited functionality. You don’t even have to register unless you want to.
How do I use ezUser?
  • Download ezUser here.
  • Add ezuser.php to your website.
  • HTML pages: put this where you want the ezUser control panel to appear
     <div id="ezuser"></div>
     <script src="ezuser.php?js=controlpanel"></script>
  • PHP pages: put this where you want the ezUser control panel to appear
     <?php
         readfile("http://your.website.com/ezuser.php?container");
     ?>
  • If you want to work with the ezUser object in your code then you can do it like this:
     require_once 'ezuser.php';
     ezUser::getContainer();
